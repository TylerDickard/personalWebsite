Work Experience

During my time at this internship I worked in the End User Engineering team in the Infrastructure Engineering Department. The project I was tasked with for the summer was Laptop/Hardware Refresh Automation. In short, I looked at the current process used at Medical Mutual and found ways to improve the efficiency, and automate tasks that were repetitive. For my individual projects I focused on two painpoints of the project, which were: large outlook datafiles taking an extremely long time to transfer for remote workers, and the logging and management of the refresh tickets was extremely tedious and several parts of the process can be automated. Here is a quick description of what I did:
  • Outlook Data File Copy Script
    • This script was able to automatically detect the file in two common save locations.
    • When the file was detected, it had the capability to split files, compress them, upload to OneDrive, and then decompress and recreate the origional File
  • Hardware Refresh Build Log Manager
    • This project was a web application, which relies upon HTML5, CSS, Javascript, jQuery, and Bootstrap 5.
    • There were tables which had the ability to add, edit, delete, and select rows of data. To implement this I used jQuery's DataTables plugin.
    • This application was able to generate labels needed for managing the refreshes physically.
    • Also, one of the key features of this project is that it had the ability to query a SQL database to pull software information needed to compare the old devices software to the new devices installed software.
